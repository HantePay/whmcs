Hantepay whmcs支付插件使用说明：

1、把gateways文件夹下面的文件上传到whmcs安装目录里面的modules/gateways/目录下。

2、开启支付接口方法：后台--》系统设置--》--》付款--》支付接口--》启用模块那里找到【Hantepay xx支付】启用。

3、为开启的支付接口配置ApiToken、商户号、以及门店编号

注意：

订单货币单位为CNY（人名币时，金额需大等于￥0.1）；
